# Changelog

All notable changes to this project will be documented in this file.

## [1.0.12] - 2024-05-06
### Added
- Production-grade Stocks MCP server with 8 tools.
- Shared utilities for retry logic, caching, and browser headers.
- Normalized response schema for all tools.
- Comprehensive README with client configuration examples.
- Examples and documentation directories.

## [1.0.11] - 2024-05-06
### Added
- Banking MCP server with IFSC and UPI validation tools.

## [1.0.10] - 2024-05-05
### Added
- Railways MCP server with PNR and train search tools.
- RTO MCP server for vehicle registration lookups.

## [1.0.0] - 2024-05-01
### Added
- Initial release with GST server.
- Basic BaseMCPServer implementation.
